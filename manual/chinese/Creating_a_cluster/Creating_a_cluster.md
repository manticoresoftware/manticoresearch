# Manticore 集群

Manticore Search 是一个高度分布式的系统，提供了创建高可用且可扩展搜索数据库所需的所有组件。这包括：
* 用于分片的 [分布式表](../Creating_a_table/Creating_a_distributed_table/Creating_a_distributed_table.md)
  * 用于高可用性的 [镜像](../Creating_a_cluster/Remote_nodes/Mirroring.md)
  * 用于可扩展性的 [负载均衡](../Creating_a_cluster/Remote_nodes/Load_balancing.md)
* 用于数据安全的 [复制](../Creating_a_cluster/Setting_up_replication/Setting_up_replication.md)

Manticore Search 在集群设置方面提供了极大的灵活性。没有限制，因此您可以根据需求设计您的集群。只需了解上述工具并使用它们来实现您的目标。
