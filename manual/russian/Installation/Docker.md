# Установка Manticore Search как Docker-образа

Docker-образы Manticore Search доступны публично на [Docker Hub](https://hub.docker.com/r/manticoresearch/manticore/), собраны из [репозитория Manticore Search docker на GitHub](https://github.com/manticoresoftware/docker).

Чтобы получить образ Manticore, выполните следующую команду:

```bash
docker pull manticoresearch/manticore
```

Для получения дополнительной информации об использовании Manticore в Docker смотрите раздел [Использование Manticore в Docker](../Starting_the_server/Docker.md).
