# KILL

<!-- example KILL -->
```sql
KILL <query id>
```

`KILL` завершает выполнение запроса по его ID, который вы можете найти в [SHOW QUERIES](../Node_info_and_management/SHOW_QUERIES.md#SHOW-QUERIES).

<!-- request SQL -->
```sql
mysql> KILL 4;
Query OK, 1 row affected (0.00 sec)
```
