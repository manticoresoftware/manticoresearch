# Комментарии

Файл конфигурации Manticore Search поддерживает комментарии, которые помогают предоставить пояснения или заметки внутри файла конфигурации. Символ `#` используется для начала комментария. Вы можете разместить символ комментария либо в начале строки, либо встроенно внутри строки.

При использовании комментариев будьте осторожны при включении символа `#` в настройки токенизации символов, так как всё, что следует после него, будет проигнорировано. Чтобы этого избежать, используйте UTF-8 код для `#`, который равен U+23.

Если вам нужно использовать символ `#` внутри вашего файла конфигурации, например, в учетных данных базы данных в описаниях источников, вы можете экранировать его с помощью обратного слэша `\`. Это позволяет включать символ `#` в настройки без его интерпретации как начала комментария.
