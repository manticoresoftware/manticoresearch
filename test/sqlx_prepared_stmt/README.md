Rust prepared statement test (sqlx)

Usage:
  cargo run --quiet

Environment:
- MANTICORE_HOST (default 127.0.0.1)
- MANTICORE_PORT (default 9306)
- MANTICORE_USER (default empty)
- MANTICORE_PASS (default empty)
- MANTICORE_DB (default empty)
- MANTICORE_ITERATIONS (default 1000)
- MANTICORE_WORKERS (default 4)
- MANTICORE_WORKER_OPS (default 200)
- MYSQL_COMPAT (set to 1 to run against MySQL)
